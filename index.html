About Me

Halo! Nama saya Indah Purnama, saya adalah seorang pelajar yang bersemangat dan berbakat dalam dunia desain kreatif. Meskipun dalam proses belajar, saya mengembangkan minat mendalam dalam desain grafis, UI/UX, dan ilustrasi. Di sela - sela tugas sekolah, saya menyempatkan waktu untuk menciptakan berbagai proyek kreatif. Dari membuat mockup untuk aplikasi presensi hingga membuat desain UI Landing Page, saya selalu berusaha memberikan sentuhan kreatif dalam setiap proyek yang saya kerjakan.

Dalam portfolio ini, saya berbagi beberapa proyek desain saya yang mencerminkan perjalanan dan perkembangan kreatif saya sebagai seorang pelajar. Saya berharap portfolio ini dapat memberikan gambaran tentang minat dan potensi saya sebagai seorang desainer masa depan. Terima kasih telah memberikan kesempatan untuk berbagi portfolio saya meskipun saya masih seorang pelajar. Saya sangat bersemangat untuk terus belajar dan mengasah kemampuan desain saya untuk mencapai tujuan karier yang lebih tinggi di masa depan.
